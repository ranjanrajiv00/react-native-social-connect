# React-Native Social-Connect

#### Get Started
```sh
$ npm install
```

#### To start the api

Run the following command

```sh
$ npm run serve
```

#### To start the expo app

Run the following command

```sh
$ npm start
```

#### To run on iOS simulator/device

Run the following command

```sh
$ npm run ios
```

#### To run on android emulator/device

Run the following command

```sh
$ npm run android
```
